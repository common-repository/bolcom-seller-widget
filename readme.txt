Show your second hand stuff from bol.com in this widget.

== Description ==

This widget shows your second hand stuff that you've added to bol.com yourself though your seller account on bol.com. It needs your seller-id and a bol.com Open API key to work.

Start promoting your second hand goods from bol.com on your Wordpres site now!

= Note =

Application Seller Widget is an independent application developed by derkmdt, with the approval of bol.com. The development of the Seller Widget is not initiated, developed or published by bol.com. Any questions regarding this plugin you can ask on the Plugin page on http://wordpress.org/plugins/bolcom-seller-widget/ under Support.

== Installation ==

1. Upload plugin directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
2. Go to the widgets
3. Add the settings for your Open API key and fill in your bol.com seller account id. Links are shown in the settings screen.
